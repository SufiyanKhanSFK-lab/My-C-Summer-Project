# My-C-Summer-Project
I am a mechanical engineering student learning C++ for the first time during my summer vacation. This repository holds the 3 basic projects I am building to practice my coding.
##  My 3 Projects

### 1. Stress and Safety Calculator
* *What it does:* Calculates stress (Force / Area) and tells you if a material is safe or will break.
* *C++ Basics Learned:* Input/Output (cin/cout), math operators, and if/else statements.

### 2. Fluid Flow Calculator
* *What it does:* Calculates the Reynolds Number to check if a fluid flow is laminar or turbulent.
* *C++ Basics Learned:* Loops (while/for) and writing simple functions.

### 3. Basic Arduino Project
* *What it does:* Connects C++ code to real hardware to blink lights and run a small motor.
* *C++ Basics Learned:* Microcontroller programming basics.

##  Goal
To learn the basics of coding so I can use it later in my engineering career.
